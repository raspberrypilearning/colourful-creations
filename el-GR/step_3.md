## Ένα λεξικό χρώματος

Η χρήση κωδίκων hex κώδικα είναι πραγματικά ευέλικτη, αλλά είναι δύσκολο να θυμηθούμε.

Όπως ίσως ήδη γνωρίζετε, ένα λεξικό σας επιτρέπει να αναζητάτε μια λέξη και να βλέπετε το νόημά της. Στην Python, ένα λεξικό είναι ακόμα πιο ευέλικτο από αυτό που - σας επιτρέπει να αναζητήσετε μια τιμή για οποιοδήποτε 'κλειδί' στο λεξικό.

Ας δημιουργήσουμε ένα λεξικό για χαρτογράφηση από ονόματα χρωμάτων φιλικά προς τον άνθρωπο (κλειδιά) με έξι κωδικούς (τιμές) φιλικούς προς τον υπολογιστή.

+ Ένα λεξικό περιέχεται σε σγουρά αγκύλια.
    
    Δημιουργήστε ένα κενό λεξικό που ονομάζεται `χρώματα`:
    
    ![screenshot](images/colourful-dict.png)

+ Επιλέξτε δροσερά ονόματα για τα χρώματα σας και επεξεργαστείτε τη γραμμή `χρώματα =` για να προσθέσετε καταχωρήσεις στο λεξικό για αυτά.
    
    Ακολουθεί ένα παράδειγμα λεξικού χρώματος:
    
    ![screenshot](images/colourful-colours.png)
    
    Ένα τετραγωνίδιο `:` χωρίζει το κλειδί (όνομα χρώματος) από την τιμή (hex code.) Χρειάζεστε ένα κόμμα `,` μεταξύ κάθε ζεύγους κλειδιών: τιμής στο λεξικό.

+ Τώρα δεν χρειάζεται να θυμάστε τους δεκαεξαδικούς κωδικούς, μπορείτε απλά να τις αναζητήσετε στο λεξικό.
    
    Προσαρμόστε τον παρακάτω κώδικα για να χρησιμοποιήσετε τα ονόματα χρωμάτων:
    
    ![screenshot](images/colourful-entries.png)
    
    Το κλειδί μπαίνει μέσα στα αγκύλια [] 'μετά το όνομα του λεξικού.

+ Τώρα μπορείτε να ενημερώσετε τον κώδικα σας για να αναζητήσετε χρώματα στο λεξικό:
    
    ![screenshot](images/colourful-use.png)

+ Δοκιμάστε τον κώδικα για να βεβαιωθείτε ότι το κείμενό σας εξακολουθεί να εμφανίζεται σωστά
